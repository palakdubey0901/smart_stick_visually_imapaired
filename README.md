 🦯 Smart Stick for Visually Impaired People
 
📌 Description
This project is a **Smart Stick designed for visually impaired people** to help them navigate safely and independently.
It uses an **ultrasonic sensor** to detect obstacles and alerts the user through a buzzer.

⚙️ Components

* Arduino Nano
* Ultrasonic Sensor (HC-SR04)
* Buzzer Module
* Battery
* Jumper wire
* Stick/Frame

 🚀 Features

* Obstacle detection
* Real-time sound alert
* Lightweight and portable
* Easy to use


 🛠️ Working

The ultrasonic sensor continuously measures the distance between the stick and nearby objects.
When an obstacle comes within a predefined range, the buzzer starts beeping to alert the user.
